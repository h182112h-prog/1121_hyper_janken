# ①課題名
エヴァじゃんけん＋追加機能：ゲーム終了後、リツコ博士からのFB（OpenAIのAPI使用）

## ②課題内容（どんな作品か）
- エヴァンゲリオンの世界観でじゃんけんゲーム
- 先にHPが0になった方が負け
- シンクロ率400%で「覚醒」し、3ターン無敵状態になる
- ゲームのログをもとに、OpenAIのAPIが分析し、赤木リツコの口調でFBが出力される

## ③アプリのデプロイURL
デプロイしている場合はURLを記入（任意）

## ④アプリのログイン用IDまたはPassword（ある場合）
なし

## ⑤工夫した点・こだわった点
- エヴァンゲリオンの世界感となるような画像・フォント選定
- 作中で重要な演出となる「覚醒」をどうゲーム上の設定に落とし込むかの模索
- 結果、1勝すると「シンクロ率」が100%上昇、負けると50%減少し、400%で3ターン無敵となる設定とした
- バトルのログを1つずつ記録させ、そのログをnode.jsを使用してOpenAIに送り、FBを出力

## ⑥難しかった点・次回トライしたいこと（又は機能）
- 「覚醒」状態における無敵モード移行へのif分岐
- 地味に、アイテムが多い中で画面に100%表示させる調整
- とにかく生成AIを使用してみたく、AIに頼りながら模索し、FBを出力できるように苦闘した。
